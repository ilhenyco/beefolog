# beefolog

Telegram-бот "Бифы в интернете" — проводник по выбору стейка и этапам приготовления.

## Запуск

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env` и заполните токен бота.
4. Запустите:

```bash
python main.py
```

## Переменные окружения

- `BOT_TOKEN` — токен Telegram-бота.
- `CHANNEL_URL` — ссылка на ваш TG-канал (по умолчанию `https://t.me/your_channel`).
